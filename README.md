# Exercice CI/CD

## init project

npm install

add a `.env` file outside of the `./src` directory with your DB variables ( follow the `.env.dist` fila as an example)

```json
{
    "DB_HOST=": "127.0.0.1",
    "DB_USER=": "<db user>",
    "DB_PASSWORD=": "<db password>",
    "DB_NAME=": "<db name>"
}

```
